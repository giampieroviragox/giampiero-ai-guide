Genera qualsiasi risorsa con Claude Code in 3 minuti

Giampiero · Giugno 2026 · 4 min di lettura

Il problema con la maggior parte dei contenuti generati dall'AI è che sono generici. Chiedi a ChatGPT di scrivere una lead magnet e ottieni lo stesso output anonimo che ottiene chiunque altro — nessuno la scarica, nessuno la condivide, finisce in una cartella di Google Drive a prendere polvere.

Questo sistema funziona diversamente. Prima di generare qualsiasi cosa, Claude ricerca cosa sta funzionando davvero nella tua nicchia in questo momento — post reali, dati di engagement reali, formati che le persone stanno salvando e condividendo. Poi usa quei pattern per costruire la tua risorsa.

Le migliori lead magnet non vengono inventate da zero. Si modellano su ciò che ha già dimostrato di funzionare. Questo sistema automatizza il passaggio di ricerca, così ogni risorsa che crei è ancorata a dati reali, non a supposizioni.

Cosa imparerai

  • Installare la skill /last30days — la ricerca di cosa performa davvero nella tua nicchia
  • Salvare la ricerca come skill riutilizzabile — così Claude ricorda i pattern vincenti per sempre
  • Generare PDF, landing page, checklist — qualsiasi documento in meno di 60 secondi

Passo 1: installa la skill di ricerca

La skill /last30days è uno strumento open-source gratuito per Claude Code che trasforma Claude in un agente di ricerca. Scansiona Reddit, X e il web per contenuti pubblicati negli ultimi 30 giorni e restituisce metriche di engagement, i performer principali e i pattern ricorrenti.

Apri il terminale ed esegui questo comando:

Terminale — esegui questo comando
git clone https://github.com/mvanhorn/last30days-skill.git ~/.claude/skills/last30days

La skill è ora disponibile in ogni sessione di Claude Code. Verificala aprendo Claude Code e digitando /last30days — dovrebbe autocompletarsi.

Passo 2: aggiungi le API key (opzionale ma consigliato)

La skill funziona subito con la ricerca web. Ma con le API key ottieni accesso a dati di engagement reali — conteggi di upvote, rapporti di like, numeri di condivisioni. Questa è la differenza tra "ho trovato alcuni post" e "ho trovato il top 1% dei post".

Terminale — configura le API key
mkdir -p ~/.config/last30days
cat > ~/.config/last30days/.env << 'EOF'
OPENAI_API_KEY=sk-your-key-here
XAI_API_KEY=xai-your-key-here
EOF
chmod 600 ~/.config/last30days/.env

Non servono entrambe le key. Anche una sola migliora notevolmente la qualità della ricerca. Se devi sceglierne una sola, prendi quella xAI — X ha i dati in tempo reale più freschi.

Passo 3: ricerca il tuo tipo di documento

Prima di generare qualsiasi cosa, devi sapere cosa funziona. Questo è il passaggio che la maggior parte delle persone salta — ed è l'unico motivo per cui le loro risorse sottoperformano. Di' a Claude che tipo di risorsa vuoi creare, essendo specifico sulla tua nicchia e il tuo pubblico.

Prompt di esempio per Claude Code
/last30days best performing lead magnets for coaches
/last30days top checklists for SaaS onboarding
/last30days most shared PDF guides in fitness
/last30days what setup guides get the most saves on Reddit

Claude restituisce dati di engagement reali: quali formati ottengono più salvataggi, quali titoli generano clic, quale struttura usano i migliori performer, e quanto sono lunghe le risorse con le migliori performance.

Passo 4: salva la ricerca come skill permanente

Questa è la mossa vincente che la maggior parte delle persone si perde. Dopo che Claude ha fatto la ricerca, puoi salvare quei risultati come skill permanente. Questo significa che ogni futura risorsa che generi usa automaticamente questi pattern — senza doverli ricercare ogni volta.

Prompt — salva la ricerca come skill
Salva questa ricerca come skill in modo che ogni volta che genero un [PDF/guida/checklist], usi questi pattern automaticamente.

Claude crea un file skill in ~/.claude/skills/ che contiene i risultati della ricerca, le regole di formattazione e i pattern strutturali. Non sei limitato a una sola skill di ricerca: puoi crearne di separate per diversi tipi di risorse — PDF, landing page, sequenze email, checklist.

Passo 5: genera qualsiasi risorsa

Ora sei pronto a generare. Di' semplicemente a Claude cosa ti serve in italiano semplice. Grazie alle skill di ricerca che hai salvato, Claude sa già come appaiono le risorse ad alte performance nella tua nicchia. L'intero processo dal prompt al PDF finito richiede meno di 60 secondi.

Prompt di generazione — esempi
Genera una guida PDF per il mio workshop su Claude Code
Crea una checklist per lanciare uno store Shopify
Costruisci una landing page per la mia cheatsheet gratuita
Scrivi una guida di riferimento rapido sul prompt engineering

Lo stack di automazione completo

Una volta che hai confidenza con le basi, puoi concatenare tutto in un singolo flusso di lavoro: ricerca con /last30days, salva come skill, genera la risorsa, poi distribuisci su Google Drive, crea una landing page e pubblica su Notion — tutto dalla stessa sessione Claude.
