I 5 connector Claude che uso ogni giorno (e perché ho eliminato gli altri 100)

Giampiero · Giugno 2026 · 6 min di lettura

Ho installato oltre 100 tra MCP e skill per Claude. Poi ho fatto un'analisi brutale: quanti di questi apro davvero ogni giorno? La risposta era 5. Il resto era rumore — strumenti eccitanti nei tutorial, poi mai più aperti.

Questa guida è lo stack che è sopravvissuto alla pulizia. Cinque connector che lavorano insieme per creare contenuti, gestire idee e automatizzare le parti più lente del flusso di lavoro. Non sono i più famosi, ma sono quelli che girano senza fermarsi.

Il quarto della lista è quello che definirei impossibile da togliere. Capirai perché quando ci arrivi.

1. kie.ai — immagini e video senza uscire da Claude

kie.ai è un MCP server che connette Claude a GPT Image 2 di OpenAI e ai modelli video Seedance. Il risultato: puoi generare immagini fotorealistiche e video direttamente dalla chat, senza aprire altri strumenti.

Il costo è fisso: $0,06 per immagine. Nessuna sorpresa, nessun tier di qualità a pagamento separato. La stessa API sblocca anche 290+ modelli tra cui Flux, Veo e Suno.

Quando usarlo: foto prodotto, creatività per ads, copertine, qualsiasi elemento con testo sovrapposto (kie.ai gestisce il testo on-image meglio delle alternative gratuite). Per bozze veloci e contenuti social usa invece un MCP gratuito come Nano Banana — poi passi a kie.ai per i finali.

Prompt di partenza — kie.ai

Genera una foto prodotto di [prodotto] su sfondo bianco pulito con il testo "Edizione Limitata" nell'angolo — fotorealistico, alta risoluzione.

Come installare: crea un account su kie.ai, genera una API key, aggiungi il blocco MCP al tuo file di configurazione Claude (claude_desktop_config.json) e riavvia Claude.

2. Notion MCP — Claude che controlla la tua workspace

Notion MCP dà a Claude accesso completo alla tua workspace Notion: crea pagine, interroga database, aggiorna proprietà, aggiunge blocchi. Non è un'integrazione leggera — Claude opera direttamente sulle tue strutture dati.

In pratica sostituisce il copia-incolla manuale da Claude a Notion, gli aggiornamenti manuali delle proprietà e l'AI nativa di Notion (che è molto più limitata).

Prompt di partenza — Notion MCP

Trova tutti i task nel mio Notion Tasks DB con stato "In Progress". Raggruppali per progetto e scrivi un riassunto di stato che posso copiare nel mio brief mattutino.

Consiglio pratico: la prima volta che dai a Claude accesso agli ID dei tuoi database Notion, salvali in un file duraturo (ad esempio ~/.config/notion-content/config.json). Da quel momento ogni prompt che tocca Notion gira senza dover ricordare gli ID.

Come installare: segui la documentazione ufficiale su developers.notion.com/docs/mcp.

3. Vercel Agent Browser — un browser costruito per gli agenti AI

Il browser integrato di Claude è stato aggiunto come ripensamento. Vercel ha costruito un browser pensato apposta per gli agenti AI: gestisce meglio i login, i form complessi, i siti dinamici e i task web multi-step.

Sostituisce il browser_tool nativo di Claude, lo scraping manuale e le API a pagamento per il web scraping.

Prompt di partenza — Vercel Agent Browser

Accedi a [nome account] e scarica gli ultimi 30 giorni di [tipo di dati] in un CSV. Se incontri un CAPTCHA, fermati e dimmi cosa mostra.

Quando usarlo: dashboard dietro login, form multi-step, scraping di dati competitivi da siti complessi.

Come installare: cerca "Vercel Agent Browser MCP" nel tuo Claude desktop o nella configurazione MCP di VS Code.

4. last30days — il connector che non riesco a togliere

Questo è il quarto della lista e quello che definirei impossibile da togliere. last30days ricerca qualsiasi argomento su 8 piattaforme in un unico prompt: Reddit, X, YouTube, TikTok, Instagram, Hacker News, Polymarket e il web aperto.

Restituisce i risultati classificati per engagement. Smetti di scrivere contenuti su quello che pensi interessi al tuo pubblico e cominci a usare dati reali su cosa viene davvero condiviso in questo momento.

Velocità: una ricerca completa su tutti e 8 le piattaforme richiede 2-5 minuti per un singolo argomento.

Prompt di partenza — last30days

/last30days AI per il fitness — concentrati sugli hook virali, salva i top 10 nella mia inspiration library

Perché è la base di tutto: ogni altro connector dello stack produce output migliori se alimentato con dati freschi di tendenza. last30days è l'input. kie.ai, Hyperframes, persino l'organizzazione in Notion — tutti producono risultati più precisi quando vengono puntati sui dati di last30days.

Come installare: github.com/mvanhorn/last30days-skill — clona il repo in ~/.claude/skills/ e riavvia Claude.

5. Hyperframes — editing video da un prompt

Hyperframes è il framework open-source di HeyGen per la composizione video. Claude scrive il video direttamente in HTML/CSS, poi lo renderizza in MP4. Didascalie, animazioni, transizioni, voiceover, effetti sincronizzati con la musica — tutto da un unico prompt.

Cosa sostituisce: Adobe Premiere ($22,99/mese), CapCut Pro, editor video freelance ($300+/video).

Curva di apprendimento: c'è, ed è reale. La prima composizione richiede ore. Alla terza sei a 20 minuti.

Prompt di partenza — Hyperframes

Crea una composizione Hyperframes di 30 secondi con: voiceover da questo script, didascalie in stile karaoke, overlay frosted-glass al 60% di alpha, tagli sincronizzati al beat ogni 2 secondi. Renderizza in 1080p verticale.

Come installare: github.com/heygen-com/hyperframes — clona il repo, esegui npm install e segui il README. Poi clona il repo anche in ~/.claude/skills/.

Come installare tutti e 5 in un'unica configurazione

Skill (last30days, Hyperframes): si installano clonando il repo in ~/.claude/skills/. Non serve configurazione MCP. Claude le carica automaticamente al riavvio.

MCP server (kie.ai, Notion, Vercel Agent Browser): ogni MCP ha un blocco JSON sulla sua pagina di installazione. Aggiungilo al blocco mcpServers nel tuo claude_desktop_config.json e riavvia Claude.

01 Clona i repo di last30days e Hyperframes direttamente in ~/.claude/skills/. Riavvia Claude — si caricano automaticamente.
02 Aggiungi i blocchi JSON di kie.ai, Notion MCP e Vercel Agent Browser al blocco mcpServers del tuo file di configurazione Claude. Riavvia.
03 Verifica: scrivi /last30days per confermare che la skill sia caricata. Scrivi list MCP tools per vedere kie-image, notion e vercel-agent-browser tra gli strumenti disponibili.

La regola dei 14 giorni

Se non hai usato un connector negli ultimi 14 giorni, eliminalo. Lo stack che usi davvero è piccolo. Lo stack che hai installato è enorme. Chiudere questo divario rende Claude più veloce e le tue decisioni più chiare.

Cosa vale la pena eliminare per primo: MCP generici per la "produttività" che avvolgono API che non usi, browser MCP precedenti alla versione Vercel, tool di image generation tranne i 2 che apri davvero, qualsiasi cosa il cui README GitHub sia più lungo della funzionalità effettiva.
