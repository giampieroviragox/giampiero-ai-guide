5 MCP per creare contenuti con Claude: installazione e prompt pronti

Giampiero · Giugno 2026 · 6 min di lettura

Gli MCP (Model Context Protocol) sono plugin che danno a Claude accesso a strumenti e servizi esterni. Quando ne colleghi uno, Claude non si limita a parlarne — accede allo strumento e lo usa per te. È la differenza tra un assistente che ti spiega come fare e uno che lo fa al posto tuo.

Per la creazione di contenuti esistono cinque MCP che cambiano completamente il flusso di lavoro: puoi creare notebook di ricerca, video animati, estrarre trascritti da qualsiasi video social, progettare interfacce e generare componenti React — tutto senza uscire da Claude.

Questa guida include il blocco JSON di configurazione e un prompt di partenza per ognuno dei cinque. Copia, incolla, riavvia Claude.

MCP 1 — NotebookLM: notebook di ricerca generati da Claude

Questo MCP permette a Claude di creare e gestire notebook su Google NotebookLM. Puoi aggiungere fonti, generare podcast, infografiche, mappe mentali, slide e guide di studio da qualsiasi contenuto — il tutto da una singola conversazione.

Installazione:

npm install -g notebooklm-mcp # Aggiungi al blocco mcpServers in ~/.claude.json: "notebooklm-mcp": { "command": "notebooklm-mcp" }

Prompt di partenza — NotebookLM

Crea un notebook NotebookLM da questi 5 URL di articoli del blog. Genera una panoramica audio e un'infografica che riassume i temi principali.

MCP 2 — Remotion: video animati programmatici da codice

Remotion è il framework per creare video come se fossero componenti React. Questo MCP dà a Claude accesso a tutto il framework: motion graphics, testo animato, visualizzazioni dati — renderizzati come file video reali.

Installazione:

# Aggiungi al blocco mcpServers in ~/.claude.json: "remotion-documentation": { "command": "npx", "args": ["-y", "@remotion/mcp@latest"] }

Prompt di partenza — Remotion

Crea un'animazione Remotion di 15 secondi con un titolo che appare in dissolvenza, tre punti elenco che compaiono uno alla volta e una CTA finale. Sfondo scuro, testo bianco, accento blu.

MCP 3 — SupaData: trascritti e dati da qualsiasi video social

SupaData permette a Claude di estrarre trascritti da YouTube, TikTok, Instagram, X e Facebook. Ma non solo il testo — anche i metadati: numero di visualizzazioni, like, commenti. Sai non solo cosa dice un video, ma quanto ha performato.

Usalo per smontare i contenuti dei competitor, trovare hook che funzionano e capire cosa rende un video virale nel tuo settore.

Installazione:

# Ottieni l'API key su supadata.ai # Salva in ~/.config/supadata/.env: SUPADATA_API_KEY=la_tua_chiave_qui # Usa tramite la skill /transcript in Claude Code

Prompt di partenza — SupaData

Estrai il trascritto da questo video [URL]. Analizza la struttura dell'hook, identifica i punti chiave e dimmi cosa ha reso questo video efficace.

MCP 4 — Stitch: schermate UI e design system da descrizioni testuali

Stitch permette a Claude di generare schermate UI complete, design system e mockup visivi da una descrizione in linguaggio naturale. Descrivi una pagina e Stitch la costruisce con layout e gerarchia visiva corretti.

Installazione:

# Aggiungi al blocco mcpServers in ~/.claude.json: "stitch": { "command": "npx", "args": ["@_davideast/stitch-mcp", "proxy"], "env": { "STITCH_API_KEY": "la_tua_chiave" } }

Prompt di partenza — Stitch

Crea una schermata landing page per il download di un PDF gratuito. Includi titolo, 4 value proposition, form di raccolta email e anteprima del PDF. Tema dark moderno, accento blu.

MCP 5 — Magic (21st.dev): componenti React pronti per la produzione

Magic permette a Claude di generare componenti React + Tailwind pronti all'uso da semplici descrizioni. Pulsanti, form, dashboard, barre di navigazione, tabelle prezzi, intere sezioni di pagina.

Installazione:

# Aggiungi al blocco mcpServers in ~/.claude.json: "magic": { "command": "npx", "args": ["-y", "@21st-dev/magic@latest"], "env": { "API_KEY": "la_tua_chiave_21st" } }

Prompt di partenza — Magic

Genera un componente pricing table con 3 livelli, checkmark per il confronto delle funzionalità e un badge "Più popolare" in evidenza. Stile moderno pulito, angoli arrotondati.

Concatenare gli MCP: il vero vantaggio

Il potere reale sta nell'usare più MCP insieme in un'unica conversazione Claude. Alcuni esempi concreti:

  • Estrai un trascritto con SupaData, caricalo in NotebookLM per creare una base di conoscenza, poi genera un riassunto video con Remotion
  • Usa Stitch per progettare il layout di una landing page, poi usa Magic per costruire i componenti React effettivi
  • Scarica i trascritti di 5 video dei competitor con SupaData, crea un notebook NotebookLM da tutti e cinque, genera un'infografica che confronta le loro strategie

Ogni catena gira in un'unica conversazione Claude. Nessun cambio di strumento, nessun copia-incolla tra app.
