Google Stitch + Claude Code: il designer AI integrato nel tuo progetto

Giampiero · Giugno 2026 · 4 min di lettura

Hai mai costruito un'app con Claude Code e alla fine sembrava fatta con uno strumento AI? Quel look generico, quegli stili di default, quell'assenza di personalità visiva che ti fa capire immediatamente che non c'è un designer dietro. Google Stitch risolve esattamente questo problema.

Google Stitch è una canvas di design AI che genera interfacce complete a partire da prompt testuali, screenshot o URL. Crea layout, palette colori, tipografia e componenti interattivi. Ogni elemento è modificabile. I prototipi sono immediati. Le anteprime responsive funzionano su tutti i device.

Quando colleghi Stitch a Claude Code tramite il suo MCP, Claude diventa un designer full-stack. Può creare tavole di design, generare sistemi di design completi e ridisegnare progetti esistenti — tutto da un singolo prompt.

Setup: collega Stitch a Claude Code

01Vai su Google Stitch e crea un account. Poi entra in Impostazioni, scorri alla sezione API e crea una chiave API. Copiala — ti servirà nel prossimo passo.
02Vai alla pagina di documentazione dello Stitch MCP e trova il comando di installazione per Claude Code. Copialo e incollalo in Claude Code insieme alla tua API key. Claude configurerà il server MCP per te.
03Riavvia Claude Code e vedrai Stitch connesso come strumento disponibile. Da questo momento Claude può elencare i tuoi progetti Stitch, recuperare singole schermate, leggere l'HTML e il CSS dei tuoi design e accedere al tuo sistema di design — tutto senza uscire da Claude Code.

Come usarlo

Design da zero. Descrivi come vuoi che appaia la tua app e Claude userà Stitch per creare design UI completi — layout, colori, font, componenti interattivi. Puoi anche dare a Claude degli screenshot o URL di riferimento e progetterà basandosi su quelli.

Ridisegnare progetti esistenti. Prendi un'app esistente che sembra fatta con l'AI. Dai a Claude degli screenshot o puntalo al tuo progetto. Claude usa Stitch per generare una direzione di design professionale, poi la applica al tuo codebase reale. La trasformazione richiede circa 5 minuti.

Il file design.md

Mentre Stitch genera i design, crea automaticamente un file design.md — un documento markdown che cattura l'intero sistema di design. Tipografia, palette colori, spaziatura, regole dei componenti, da fare e da non fare. Questo file è ciò che mantiene i tuoi design coerenti in tutto il progetto.

Claude Code può leggere questo file e seguirlo come riferimento per ogni modifica che fa. Man mano che il design evolve, aggiorna design.md e Claude mantiene la coerenza automaticamente.

Casi d'uso principali

  • Ridisegnare un'app vibe-coded che sembra generica trasformandola in qualcosa di professionale
  • Creare un sistema di design completo per un nuovo progetto prima di scrivere qualsiasi riga di codice
  • Generare design di web app responsive da una singola descrizione testuale
  • Costruire landing page, dashboard e interfacce SaaS che sembrano prodotti reali
  • Mantenere stili coerenti su più pagine e funzionalità
  • Creare tavole di design e prototipi da condividere con clienti o team

L'intuizione chiave

Stitch non crea solo belle immagini. Crea design reali e interattivi supportati da un sistema di design che Claude Code può leggere e seguire. Questo significa che il gap tra ciò che progetti e ciò che viene costruito scompare. Quello che vedi in Stitch è quello che Claude costruisce nella tua app.
