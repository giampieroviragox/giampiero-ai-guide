Genera qualsiasi immagine dentro Claude Code con NanoBanana

Giampiero · Giugno 2026 · 7 min di lettura

Ogni progetto ha bisogno di immagini. I post del blog vogliono una featured image. Le landing page vogliono una hero graphic. Le campagne pubblicitarie vogliono varianti creative. Le presentazioni vogliono diagrammi. E ogni volta che ne hai bisogno, esci dal workflow — apri Canva, aspetti Midjourney, o scrivi a un designer e aspetti giorni.

NanoBanana 2 elimina tutto questo. Connette i modelli di generazione immagini di Google Gemini direttamente a Claude Code come server MCP. Una volta installato, Claude ha sette strumenti per le immagini integrati. Descrivi quello che vuoi, Claude lo genera, e l'immagine finisce nella directory del progetto. Nessun cambio di app. Nessuna attesa. Nessun abbonamento separato.

La qualità dell'output è seria: risoluzione 4K, rendering accurato del testo, dieci formati diversi, e modifica multi-turno dove descrivi i cambiamenti e Claude aggiorna l'immagine istantaneamente.

Setup in 3 comandi

Passo 1 — Installa il server MCP

Terminale

npm install -g nanobanana-mcp-server

Passo 2 — Registra con Claude Code

Terminale

claude mcp add nanobanana -- node ~/.gemini/extensions/nanobanana/mcp-server/dist/index.js

Passo 3 — Imposta la chiave API

NanoBanana usa l'API Gemini di Google per la generazione delle immagini. Se hai già una chiave Gemini configurata, sei a posto. Altrimenti prendila gratuitamente da Google AI Studio e impostala come variabile d'ambiente:

Terminale

export GEMINI_API_KEY=la_tua_chiave_qui

Per verificare la connessione, apri Claude Code e chiedigli di generare un'immagine di test semplice. Se compare un'immagine nella directory del progetto, sei connesso.

Upgrade al modello Pro

Per default, NanoBanana usa Gemini Flash — veloce e buona qualità. Per output in 4K e rendering del testo migliore, imposta NANOBANANA_MODEL=gemini-3-pro-image-preview nel tuo ambiente. Qualità superiore, leggermente più lento.

I 7 strumenti immagine

Una volta connesso, Claude Code ha sette strumenti distinti. Ognuno gestisce un tipo diverso di asset visivo.

  • Genera immagini — generazione testo-immagine con controllo dello stile. 10 stili (fotorealistico, acquerello, pittura a olio, schizzo, pixel art, anime, vintage, moderno, astratto, minimalista) e 10 formati (16:9, 1:1, 4:3, 9:16 e altri). Fino a 8 variazioni contemporaneamente con illuminazione, angoli, palette o mood diversi.
  • Modifica immagini — modifica qualsiasi immagine esistente con linguaggio naturale. Dì "rendi lo sfondo più scuro" o "rimuovi il testo e aggiungi un tramonto." Modifica multi-turno — continua a raffinare fino a che non è giusta.
  • Restaura immagini — ripara e migliora foto vecchie o danneggiate. Correggi graffi, migliora la risoluzione, correggi i colori, rimuovi il rumore.
  • Genera icone — crea icone app, favicon ed elementi UI in più dimensioni (da 16px a 1024px). Stili piatto, skeumorfici, minimali o moderni. Sfondi trasparenti o colorati.
  • Genera pattern — pattern e texture seamless per sfondi, tessuti e materiali. Stili geometrici, organici, astratti, floreali o tech. Controllo della densità e della palette colori.
  • Genera diagrammi — diagrammi tecnici, flowchart, architetture, mappe di rete, wireframe, mappe mentali e diagrammi di sequenza. Stili professionali, puliti, a mano libera o tecnici.
  • Genera sequenze narrative — da 2 a 8 immagini sequenziali per tutorial, timeline o flussi di processo. Stile coerente o evolutivo nei frame. Output come file separati, griglia o layout a fumetto.

La libreria gratuita di 1.000+ prompt

Scrivere prompt per immagini di qualità da zero è difficile. Questo sito risolve il problema: oltre 1.000 prompt NanoBanana organizzati in più di 40 casi d'uso e stili. Nessuna registrazione richiesta.

youmind.com/nano-banana-pro-prompts

Come usarla: naviga per caso d'uso (thumbnail, infografiche, annunci, riprese cinematografiche, render 3D, post social) o per stile (neon cyberpunk, retrò vintage, minimal pulito, acquerello, anime). Trovato un prompt che ti piace, copialo, incollalo in Claude Code, sostituisci il soggetto e genera. Due minuti dall'idea all'immagine finita.

I casi d'uso migliori dalla libreria

  • Thumbnail YouTube — design d'impatto che attirano i click. Stili multipli dal bold e grafico al cinematografico e cupo.
  • Infografiche — visualizzazioni di idee complesse. Layout puliti con data visualization, flussi di processo e griglie comparative.
  • Creatività per annunci — foto prodotto, immagini lifestyle e grafiche promozionali. Ottimizzate per feed social, stories e display ads.
  • Post social — visual ferma-scroll per Instagram, LinkedIn e X. Grafiche con citazioni, post di annuncio e header per contenuti.
  • Render 3D di prodotti — foto prodotto fotorealistiche senza fotografo. Perfette per ecommerce, pitch deck e marketing.
  • Riprese cinematografiche — composizioni di scene di qualità cinematografica. Ottime per header di blog, sfondi per presentazioni e progetti creativi.

Workflow reali

Featured image per blog post — stai scrivendo un articolo in Claude Code? Quando hai finito, chiedi a Claude di generare una featured image basata sull'argomento. Legge il contenuto, sceglie un concetto visivo appropriato e genera l'immagine direttamente nella directory delle immagini del blog. Nessun cambio di contesto.

Hero graphic per landing page — invece di cercare foto stock, chiedi a Claude di generare un'immagine hero che corrisponda al brand e al messaggio. Genera più variazioni, scegli la migliore, è già nella cartella del progetto.

Varianti creative per A/B test — hai bisogno di cinque varianti di annuncio per l'A/B test? Generale tutte in una sessione. Stesso prodotto, stili diversi — fotorealistico, minimale, grafico bold, lifestyle, cinematografico.

Prompt da copiare subito

Thumbnail YouTube

Genera una thumbnail YouTube per un video su [IL TUO ARGOMENTO]. Testo bold, alto contrasto, composizione d'impatto. Formato 16:9. Deve sembrare qualcosa su cui si clicca.

Featured image blog

Genera una featured image per questo articolo del blog. Leggi il contenuto e crea un visual che cattura il concetto principale. Stile pulito e moderno. Paesaggio 16:9. Nessun testo sull'immagine.

Varianti creative per annunci

Genera 4 varianti creative per [IL TUO PRODOTTO]. Ognuna deve usare uno stile visivo diverso: fotorealistico, minimale, grafico bold e lifestyle. Formato quadrato (1:1) per i feed social.

Foto prodotto professionale

Genera una foto prodotto professionale di [IL TUO PRODOTTO] su sfondo neutro. Illuminazione da studio. Leggera ombra. Stile fotorealistico. Alta risoluzione.
