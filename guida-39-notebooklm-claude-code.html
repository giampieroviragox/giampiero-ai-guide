Connetti NotebookLM a Claude Code: ricerca AI senza limiti di token

Giampiero · Giugno 2026 · 7 min di lettura

NotebookLM è il miglior strumento di ricerca che Google abbia mai costruito. Trova le fonti di maggiore qualità su qualsiasi argomento, le legge tutte e crea riassunti che puoi usare davvero. Il problema è che vive in una scheda del browser: devi passare continuamente da un'app all'altra, copiare e incollare, gestire tutto manualmente.

Claude Code è il miglior assistente AI per fare lavoro concreto. Costruisce, automatizza, esegue. Ma quando ha bisogno di ricercare qualcosa, è limitato alle ricerche web e ai propri dati di addestramento — e ogni query ti costa token.

Connettili con notebooklm-py e ottieni qualcosa che nessuno dei due riesce a fare da solo: Claude Code ordina a NotebookLM di fare ricerche approfondite, poi genera riassunti audio, infografiche, slide modificabili, mappe mentali, flashcard, quiz e molto altro — tutto da un prompt, tutto senza uscire dal terminale. I server di Google fanno tutto il lavoro pesante. Zero token Claude spesi per l'elaborazione.

Setup in 3 comandi

01Installa notebooklm-py. Apri il terminale e installa il pacchetto. Ti dà un'interfaccia a riga di comando completa per NotebookLM con ogni funzione — incluse quelle che l'interfaccia web non espone.

Comando — Installazione

pip install "notebooklm-py[browser]"
playwright install chromium
02Accedi al tuo account Google. Esegui il comando di login. Si apre una finestra del browser — accedi con il tuo account Google e sei autenticato.

Comando — Login

notebooklm login
03Installa la skill per Claude Code. Questo installa una skill che insegna a Claude Code come usare ogni singolo comando di NotebookLM. Dopo questo, Claude sa creare notebook, aggiungere fonti, generare contenuti, scaricare file e molto altro — tutto in automatico.

Comando — Installa skill

notebooklm skill install

Per verificare la connessione, apri Claude Code e chiedigli di elencare i tuoi notebook. Se restituisce la tua dashboard NotebookLM, sei connesso. Se Claude Code non riesce a connettersi, esegui di nuovo notebooklm login — i cookie di autenticazione scadono periodicamente. Puoi anche eseguire notebooklm auth check --test per diagnosticare eventuali problemi.

Tutto quello che puoi creare

Una volta connesso, Claude Code ha accesso programmatico completo a NotebookLM — incluse funzioni che l'interfaccia web non espone. Ogni notebook supporta fino a 50 fonti.

  • Riassunti audio (4 formati): conversazioni in stile podcast tra due conduttori AI. Quattro formati: approfondito, breve, critico e dibattito. Tre opzioni di lunghezza. Oltre 50 lingue. Impiega circa 3-5 minuti. Perfetto per imparare in movimento.
  • Infografiche (11 stili): riassunti visivi in un'unica immagine condivisibile. 11 stili: bento grid, professional, editorial, instructional, sketch-note, scientific, bricks, clay, anime, kawaii e auto. Tre orientamenti e tre livelli di dettaglio.
  • Slide deck (PowerPoint modificabile): presentazioni complete generate dalla tua ricerca. Scarica come PDF o PPTX modificabile — l'interfaccia web offre solo PDF, ma notebooklm-py ti dà il file PowerPoint. Puoi anche revisionare singole slide con prompt in linguaggio naturale.
  • Video riassunti (9 stili visivi): whiteboard, anime, kawaii, watercolor, retro-print, heritage e paper-craft. Tre formati: esplicativo, breve e cinematografico.
  • Mappe mentali, quiz, flashcard: le mappe esportano come JSON. Quiz e flashcard esportano come JSON, Markdown o HTML — l'interfaccia web mostra solo una vista interattiva. Le tabelle dati esportano come CSV. I report generano come documenti di briefing, guide allo studio o post del blog.

3 casi d'uso reali

1. Intelligence competitiva settimanale. Ogni lunedì, di' a Claude di ricercare cosa hanno rilasciato i tuoi 3 principali concorrenti la settimana scorsa. Costruisce un notebook NotebookLM con i loro post, comunicati stampa, aggiornamenti di prodotto e menzioni social. Poi genera un riassunto audio da ascoltare durante il tragitto e un'infografica da condividere con il team. 15 minuti di tempo di ricerca sostituiti da un prompt.

2. Pacchetti di ricerca pronti per i clienti. Prima di ogni chiamata con un cliente, di' a Claude di ricercare le tendenze del settore del cliente, generare una slide deck modificabile con i risultati chiave e creare un'infografica che riassume i dati. Scarica il PPTX, aggiungi il tuo branding e presenta. Arriva a ogni riunione con ricerche personalizzate che sembrano il lavoro di giorni del tuo team.

3. Ricerca per contenuti in automatico. Stai creando contenuti su un argomento di cui non sei esperto? Di' a Claude di ricercarlo via NotebookLM — aggiungi i migliori video YouTube come fonti così legge le trascrizioni, poi genera un riassunto audio in formato dibattito per sentire entrambi i lati. Fai domande di follow-up finché capisci l'argomento. Poi chiedi a Claude di scrivere il tuo script, post del blog o newsletter — basandosi su fonti che ha davvero letto.
