Da Foglio Excel a Web App in 10 Minuti con Claude Code

Giampiero · Giugno 2026 · 5 min di lettura

Hai un foglio Excel o Google Sheets che usi ogni giorno — una lista clienti, un inventario, un budget, un tracker di progetti. Funziona, ma è rigido, anonimo, senza filtri decenti e impossibile da usare bene da mobile. Nel frattempo sai che da qualche parte esiste una versione migliore di quello strumento: un'app vera, con un'interfaccia pulita, una barra di ricerca, form per aggiungere dati, filtri per colonna.

Quella versione migliore non richiede un programmatore. Richiede Claude Code e circa 10 minuti del tuo tempo. Il processo è semplice: esporti il foglio come CSV, lo carichi in Claude Code, incolli un prompt. Claude analizza i tuoi dati, capisce cosa ci stai facendo e costruisce un'app funzionante che gira in locale sul tuo computer — privata, veloce, tua.

Ho usato una lista ospiti per un matrimonio come esempio, ma il metodo funziona con qualsiasi foglio: tracker clienti, gestione magazzino, budget familiare, lista progetti. Il prompt è identico, cambia solo la descrizione dei dati.

Cosa ti serve prima di iniziare

  • Claude Code installato (gratuito da claude.com/code)
  • Un abbonamento Claude Pro (20$/mese)
  • Il tuo foglio di calcolo — Google Sheets o Excel va benissimo
  • La skill UI-UX Pro Max installata (spiegata nel passo 1)
  • Nessuna esperienza di programmazione richiesta

Passo 1 — Installa la skill UI-UX Pro Max

Vai su GitHub e installa la skill UI-UX Pro Max. È quello che trasforma le app costruite da Claude da "funzionali ma brutte" a "sembrano fatte da un designer professionista". Senza questa skill il risultato è anonimo. Con essa, l'interfaccia è pulita, moderna e piacevole da usare.

Il link è: github.com/nextlevelbuilder/ui-ux-pro-max-skill

Apri Claude Code e digli: "Installa la skill da questo repository GitHub." Claude fa tutto il resto da solo.

Passo 2 — Esporta il foglio come CSV

Apri il tuo foglio di calcolo ed esportalo come file CSV.

  • Google Sheets: File → Scarica → Valori separati da virgola (.csv)
  • Excel: File → Salva con nome → CSV

Salvalo in una cartella facile da trovare. Questo è l'unico file di cui hai bisogno.

Passo 3 — Carica il CSV e usa questo prompt

Apri Claude Code in una nuova cartella vuota. Trascina il file CSV nella chat. Poi incolla questo prompt, sostituendo solo la parte tra parentesi quadre:

Prompt da copiare

Ti fornisco un CSV con [descrivi cosa contiene il foglio]. Voglio che tu:
1. Analizzi le colonne e capisca a cosa serve questo foglio
2. Costruisca un'app web funzionante che posso avviare in locale per visualizzare e lavorare con questi dati
3. L'app deve includere:
   - Un'interfaccia pulita e professionale (usa la skill UI-UX Pro Max)
   - Form per aggiungere, modificare ed eliminare voci
   - Filtri e ricerca su tutte le colonne principali
   - Una vista ottimizzata per mobile
4. Usa il tech stack più semplice possibile (Next.js funziona benissimo)
5. Quando è pronta, dammi i comandi esatti per avviarla sul mio computer
Fammi 2-3 domande rapide su come voglio usare questa app prima di iniziare a costruire.

Passo 4 — Avvia l'app in locale

Claude costruisce l'app e ti fornisce i comandi per avviarla. Di solito sono due righe:

Comandi di avvio

npm install
npm run dev

Apri http://localhost:3000 nel browser. Prova ad aggiungere una voce, filtra per colonna, cerca un termine. L'app gira interamente sul tuo computer, quindi è privata e istantanea.

Come migliorare l'app passo dopo passo

La prima versione è un punto di partenza, non un prodotto finito. Puoi continuare ad aggiungere funzioni semplicemente descrivendole a Claude:

  • "Aggiungi una dashboard con grafici che mostrano i trend nel tempo"
  • "Aggiungi un interruttore per la modalità scura"
  • "Colora le righe in base alla colonna Stato"
  • "Aggiungi una vista stampabile da usare su carta"
  • "Genera un report di riepilogo basato sui dati presenti"

Ogni modifica richiede circa 5 minuti. Non tocchi mai il codice. Il passo successivo, quando vuoi, è mettere l'app online per condividerla con altri — ma per ora gira in privato sul tuo computer, ed è già uno strumento completamente funzionante.
